# 🎥 Twitch 工具箱

這是一個基於 **PyQt6** 開發的桌面應用程式，整合了兩大核心功能：
**直播錄影保存** 與 **開播通知觀看**，讓你不再錯過喜愛實況主的任何精彩時刻。

---

## 📦 下載

1. 前往 [Releases](../../releases) 頁面下載最新版本的 `Twitch工具箱.exe`
2. 下載後即可直接執行，**不需要安裝 Python**

---

## 🔑 準備憑證（必做）

要使用 Twitch API 查詢頻道開播狀態，必須先建立應用程式來取得 **Client ID** 和 **Client Secret**。
以下是步驟：

1. 前往 [Twitch Developer Console](https://dev.twitch.tv/console/apps)
   （需要使用你的 Twitch 帳號登入）

2. 點擊 **「+ Register Your Application」**
   - **Name**：隨便填寫一個名稱（例如 `TwitchToolbox`）
   - **OAuth Redirect URLs**：填入 `http://localhost`
   - **Category**：選擇「Application Integration」

3. 建立後，你會看到應用程式詳細資訊
   - 複製 **Client ID**
   - 點擊 **New Secret** 產生並複製 **Client Secret**

4. 啟動 `Twitch工具箱.exe`
   - 在「開播通知觀看」頁籤中填入 **Client ID** 和 **Client Secret**
   - 點擊「更新 Token」自動產生 Access Token

⚠ 注意：
- **Client Secret 不建議在公用電腦保存**
- Token 會自動管理，到期前會自動刷新

## 🚀 功能介紹

### 📹 直播錄影保存

自動監控並錄製 Twitch 直播，使用 Streamlink 技術確保高品質錄影。

1. **新增監控頻道**
   - 輸入實況主 ID（例如：`kspksp`）
   - 點擊「新增監控」加入清單

2. **選擇畫質**
   - 支援 `best`、`1080p60`、`720p60`、`audio_only` 等選項

3. **設定存檔位置**
   - 點擊 📂 按鈕選擇錄影檔案存放資料夾
   - 錄影檔會自動以「頻道名稱_日期時間.ts」命名

4. **開始錄影監控**
   - 點擊「準備就緒 (點擊開始監控)」按鈕
   - 程式會持續監控，一旦偵測到開播就自動開始錄影
   - 錄影日誌會即時顯示錄影狀態

---

### 🔔 開播通知觀看

監控多個 Twitch 頻道，偵測到開播時自動打開瀏覽器觀看。

1. **加入頻道**
   - 在輸入框輸入 Twitch 頻道名稱（例如：`kspksp`）
   - 按下「加入」 → 頻道會出現在清單中
   - 點擊「✕」按鈕可以快速刪除頻道

2. **設定檢查間隔**
   - 可以輸入「分鐘」與「秒」作為檢查時間（例如 `0 分 30 秒`）
   - 預設為 60 秒

3. **開始監看**
   - 點擊「開始監看」 → 程式會依照設定間隔自動檢查
   - 當偵測到有頻道開播，會立刻自動開啟瀏覽器觀看
   - 右側表格會即時顯示各頻道的狀態與直播標題

4. **停止監看**
   - 點擊「監看中 (點擊停止)」即可暫停檢查

## ✨ 功能特色

- **雙功能整合**：錄影保存 + 開播通知，一個程式搞定
- **自動 Token 管理**：Access Token 到期前自動刷新
- **多頻道支援**：可同時監控多個實況主
- **自訂檢查間隔**：可設定分鐘與秒數
- **畫質選擇**：錄影支援多種畫質選項
- **系統列常駐**：關閉視窗後縮小至系統列，持續運作
- **開機自啟動**：可設定開機時自動啟動並開始監控/錄影
- **深色介面**：Twitch 風格深色主題，簡潔易用

---

## ⚠ 注意事項

- Twitch API 有頻率限制，請避免設定過短的檢查間隔（建議 ≥30 秒）
- 若使用防毒軟體，可能會誤判為未知程式，請加入白名單
- 程式會在同一場直播中 **只開啟一次瀏覽器**，避免重複彈出
- 錄影功能需要足夠的硬碟空間，請確保存檔路徑有足夠容量
- 錄影檔案格式為 `.ts`，可使用一般播放器播放或轉檔
